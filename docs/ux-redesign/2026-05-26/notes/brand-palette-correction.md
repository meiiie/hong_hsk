# Brand Palette Correction

The first UX north-star mockups overused teal/jade. That made the app feel calmer, but it drifted away from the actual Hồng HSK4 brand.

## Source Of Truth

- `public/icon.svg`
- `public/maskable-icon.svg`
- `public/og-image.png`
- `docs/branding/hong-hsk4-studio-brand-concept.png`
- `src/presentation/styles.css`

## Correct Direction

- Primary: Hồng/seal red `#9f2734`
- Strong primary: deep burgundy `#7f1d2a`
- Active lift: warm red `#d95d67`
- Soft brand surface: blush `#fff0f1`
- Page background: warm paper `#f8f3ef`
- Text: ink navy `#1f2230`
- Secondary accent only: jade `#0f766e`

## Design Rule

The product should feel like a modern study tool built around a Chinese red seal and warm paper texture. Jade can support progress, calm states, and small data accents, but primary navigation, logo presence, and main calls to action should not become teal-led.
