# Image Generation Prompt Pack

All prompts used built-in `image_gen` and were copied into this workspace after generation.

## Public Assets

- GitHub README hero: red-first product banner with red Chinese seal, app phone preview, and warm ivory paper background.
- Open Graph preview: 1200x630 social card direction for `public/og-image.png`, with red seal identity and minimal jade accents.

## UX/UI Resources

- Mobile study states: queue, recall, checking, wrong recovery, correct scheduling.
- Mock exam mode: set selection, listening, reading, writing/reorder, submit confirmation.
- Component system: buttons, navigation, cards, input states, quiz choices, chips, alerts, data cards, typography.
- Data operations: Excel/CSV import, lesson coverage, weak words, 30-day plan, data quality warnings.

## Hard Constraints

- Hồng/seal red `#9f2734` is primary.
- Deep burgundy `#7f1d2a` supports strong surfaces.
- Warm paper `#f8f3ef` and blush `#fff0f1` are the main backgrounds.
- Ink navy `#1f2230` is the primary text color.
- Jade `#0f766e` is secondary only.
- Avoid teal primary buttons, green logos, purple/blue gradients, generic SaaS dashboards, and text copied from generated images.
